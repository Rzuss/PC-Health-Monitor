Features

Everything your PC needs,
nothing it doesn't

Nine specialized tabs, each laser-focused on a single aspect of your system's health.

📊

Real-Time Dashboard

Arc gauges for CPU, RAM, and Disk with a live Health Score. Sparkline CPU history chart updates every second using LibreHardwareMonitor.

⚡

Startup Manager

See every program that launches at Windows startup. Toggle or permanently remove entries — changes go to HKCU only, no system-wide edits.

🧹

Intelligent Junk Cleaner

Scans Temp, Prefetch, browser caches, Windows Update leftovers, and more. Shows exact sizes before you clean — no surprises.

💾

Storage Analyzer

Visual breakdown of every drive. Identifies largest folders at a glance so you know exactly where your disk space went.

🚀

Boost Mode

Elevate priority of any running process for gaming or rendering workloads. One click to boost, one click to restore — fully reversible.

🖥️

System Information

Detailed hardware inventory: CPU model & cores, RAM slots and speed, GPU, motherboard, BIOS version, and OS build — all from WMI read-only.

🔧

Driver & Auto-Schedule

Audit installed drivers and schedule automatic silent cleanups via Windows Task Scheduler — runs at 2 AM with a tray notification when done.

🌐

Network Monitor

Live adapter stats, active connections, ping latency, and DNS info. Identifies which processes are consuming bandwidth right now.

⚙️

Smart Settings

Window size and position remembered between sessions. Toggle startup, tray behavior, and notification preferences — saved to a local JSON file.

How it Works

Up and running in 3 steps

1

Download the Installer

One self-contained .exe file (~80 MB). No .NET runtime installation needed — everything is bundled inside.

2

Install Without Admin

The installer places the app in your personal %LocalAppData% folder. No UAC prompt, no system changes.

3

Monitor & Optimize

Launch from the Desktop shortcut. The dashboard starts live monitoring immediately — no configuration required.

Privacy & Security

Zero network traffic.
Ever.

PC Health Monitor is a fully offline application. It reads your hardware, writes to your local Temp folders, and nothing else. Your system data never leaves your machine — not even for crash reports or analytics.

🔒 Verify the download SHA-256 hash on the Releases page to confirm you have the official binary.
Reads local hardware info (CPU, RAM, Disk) ✓ Yes — by design
Writes to Temp / junk folders (cleanup) ✓ Yes — user-initiated only
Modifies Startup registry (HKCU only) ✓ Yes — only when you click Remove
Sends data to any external server ✕ Never
